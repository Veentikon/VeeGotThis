# VeeGotThis
A shared family task & event manager.
Built with React, FastAPI, Docker, and PostgreSQL (or SQLite for MVP).

## Features (MVP)
- Shared family tasks
- Assign / take tasks
- Due dates
- Simple dashboard

## Technilogies
- Backend: Pythong + FastAPI
- Frontend: React + Typescript
- Database: SQLite (development), Postgress (deployment)
- Docker + docker-compose
